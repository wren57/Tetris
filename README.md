## 操作说明

- ※ 请确保自己的输入法为英文状态
- 开局通过 `0` 或 `1` 选择普通或地狱难度，对应不同的游戏速度和背景音效
- 小写 `wsad` 控制上下左右方向
- 小写 `k` 控制方块变向
- 每消除一层方块，游戏速度便会提升
